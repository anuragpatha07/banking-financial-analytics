# Financial Modelling & Banking Analytics using Python, SQL and PostgreSQL

This repository contains academic and practical projects focused on Financial Modelling, Banking Data Analytics, Statistical Analysis, and Database Management using Python, PostgreSQL, SQL, DBeaver, Pandas, NumPy, and Data Visualization tools.

The projects are based on real banking datasets including:

* Union Bank of India
* Central Bank of India

The work includes complete financial analysis workflows such as CSV handling, database integration, regression analysis, ANOVA, correlation analysis, financial modelling, and visualization. The research foundation and task documentation are based on the Business Analytics course project submitted at Mittal School of Business.

# Tasks Included

## Task 1 – Bibliometric Analysis using VOSviewer

Performed bibliometric network analysis to study relationships between banking research variables including CAMEL parameters, workforce productivity analytics, and governance frameworks.

## Task 2 – CSV Import in Python

Imported and cleaned financial datasets using Pandas for further financial analysis.

## Task 3 – Financial Bar Graph Visualization

Created comparative bar charts for financial variables such as Equity Capital, Deposits, and Total Assets.

## Task 4 – Descriptive Statistics

Performed statistical summaries to understand distribution and variability of financial indicators.

## Task 5 – Correlation Analysis

Generated correlation matrices and heatmaps to examine relationships among banking financial variables.

## Task 6 – Multiple Regression Analysis

Applied OLS regression using Statsmodels to evaluate relationships among financial indicators.

## Task 7 – Simultaneous Equation Model

Developed financial relationship models to study interactions between borrowings and financial strength.

## Task 8 – ANOVA Testing

Conducted One-Way ANOVA to identify statistically significant differences among financial groups.

## Task 9 – Financial Calculator

Created a Python-based financial calculator for arithmetic and ratio analysis.

## Task 10 – Regression and Visualization

Performed advanced financial analysis and graphical interpretation using Python libraries.
